If you're coding with Google's Antigravity CLI (`agy`), the raw terminal gets annoying fast (copy-pasting context, formatting prompts).

There's no official VS Code extension for it, and switching to Google's Antigravity IDE is a pain if you're hooked on VS Code (like I am).

So I built a simple helper: **Antigravity CLI Companion** 🚀

It's a fast, local UI layer over your terminal:
- `Alt + G` to turn code selections into context chips
- Clean input field (no terminal typo battles)
- Mode toggles + auto-recovers sessions

100% safe & handles input only.
